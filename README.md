# TheMovie App

TheMovie là ứng dụng Android giúp người dùng tìm kiếm thông tin về phim, xem chi tiết, và quản lý danh sách phim yêu thích.

## Features
- Tìm kiếm phim theo tên
- Xem chi tiết phim (diễn viên, thể loại, ngày phát hành)
- Quản lý danh sách phim yêu thích
- Giao diện đơn giản, thân thiện với người dùng
